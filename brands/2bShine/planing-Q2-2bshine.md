## 2. META ADS STRATEGY — CPAS SHOPEE

### Campaign Structure

```
Campaign Level: CBO (Campaign Budget Optimization)
├── Ad Set 1: COLD AUDIENCE — Interest targeting (40%)
├── Ad Set 2: WARM AUDIENCE — Engagement retargeting (35%)
└── Ad Set 3: HOT AUDIENCE — Shopee cart abandoners (25%)
```

### Targeting Breakdown

**Cold (Budget: 40%)**
```
Interest: Skincare, kecantikan, K-beauty, Somethinc, Skintific
Behavior: Shopee frequent shoppers, online beauty buyers
Age: 18–30 | Wanita | Indonesia
Exclude: Existing customers
```

**Warm (Budget: 35%)**
```
Custom Audience: IG/FB engagers 60 hari
Website visitors (jika ada pixel)
Video viewers 75% (dari konten organik)
Lookalike: 1-3% dari existing buyers
```

**Hot — Retarget (Budget: 25%)**
```
Shopee CPAS: Retarget yang klik produk tapi belum beli
Cart abandoners di Shopee
Profile visitors IG/FB 30 hari terakhir
```

### Content Angles & Creative Format

| Angle                 | Format                     | Hook                                                 |
|---                    |---                         |---                                                   |
| Problem-Solution      | UGC Video 15s              | "Kulit kusam meski rajin skincare? Ini kenapa..."    |
| Ingredient Edukasi    | Carousel/Single Image      | "Kenapa niacinamide 10% di 2bShine lebih efektif?"   |
| Before-After          | UGC Video 30s              | "30 hari pakai 2bShine, ini hasilnya jujur..."       |
| Social Proof          | Testimonial collage        | "1.200+ review bintang 5 di Shopee"                  |
| Limited Offer         | Single Image               | "Gratis ongkir + bonus serum. Hari ini aja."         |

### Benchmark Estimasi (Skincare Indonesia)

| Metric                | Benchmark          | Target 2bShine |
|---                    |---                 |---             |
| CTR                   | 1.0–2.5%           | 1.5%+          |
| CPC                   | Rp 800–2.000       | < Rp 1.500     |
| Conversion Rate       | 2–5%               | 3%             |
| ROAS                  | 2x–5x              | 2.5x+ (bulan 1) → 3x+ (bulan 3) |
| CPM                   | Rp 15.000–35.000   | < Rp 25.000    |

### Testing Framework — Bulan 1

```
Step 1: Testing Phase (Minggu 1-2)
→ Jalankan 5-8 creative berbeda @ Rp 50K/hari per creative
→ Ukur CTR, CPC, dan ROAS setiap 3 hari
→ Budget total testing: Rp 2-3jt di bulan 1

Step 2: Identify Winners (Minggu 2-3)
→ Winning criteria: CTR > 2%, ROAS > 2.5x
→ Pause semua yang tidak memenuhi kriteria setelah Rp 150K spend

Step 3: Scale (Bulan 2-3)
→ Naikkan budget winning ads MAX 30%/hari
→ Jangan langsung double budget — tunggu 3 hari dulu
→ Duplicate ke audience baru (lookalike 3-5%)
→ Buat variasi creative dari winning angle
```

## 3. TIKTOK AFFILIATE STRATEGY

### Sistem Kerja 100–130 KOL

> **Note internal:** Target 100–130 KOL (bukan 350). Manage 350 KOL membutuhkan minimal 1 ops person full-time + tools mahal. 130 KOL bisa dihandle 1 PIC dengan spreadsheet + DM yang terstruktur.

**Tier Structure:**

| Tier     | Followers  | Jumlah KOL | Komisi | Budget/KOL            |
|---       |---         |---         |---     |---                    |
| Nano     | 1K–10K     | 80–100     | 10–15% | Product only (Rp 30K HPP) |
| Micro    | 10K–100K   | 20–25      | 12–18% | Product + Rp 50–200K  |
| Mid-tier | 100K–500K  | 3–5        | 15–20% | Product + Rp 500K–1jt |

**Budget TikTok Affiliate Rp 11.250.000 — Breakdown Detail:**
```
Product seeding nano KOL (90 unit @ Rp 30K): Rp 2.700.000
Fee micro KOL (22 KOL avg Rp 75K): Rp 1.650.000
Product seeding micro (22 unit @ Rp 30K): Rp 660.000
Mid-tier fee (4 KOL avg Rp 750K): Rp 3.000.000
Boost konten affiliate winner via TikTok Ads: Rp 2.500.000
Admin tools + tracking: Rp 740.000
---
Total: Rp 11.250.000
```

> **Komisi (10–18% dari sales):** TIDAK diambil dari budget ini, tapi HARUS dihitung dalam evaluasi ROAS riil.

**Alur Rekrutmen → Scaling:**

```
1. RECRUIT (Minggu 1-2)
   ↓ Cari via TikTok search: "skincare review" "skincare lokal"
   ↓ DM template: produk gratis + komisi 15% untuk 1 video
   ↓ Target: 80 nano KOL onboard bulan 1

2. BRIEF (Sebelum kirim produk)
   ↓ Brief 1 halaman: do's & don'ts, hook suggestion, hashtag wajib
   ↓ Minta konten dalam 7 hari setelah terima produk
   ↓ Jika tidak posting dalam 7 hari → send reminder script

3. CONTENT (Minggu 3-4)
   ↓ KOL posting konten
   ↓ Gunakan link affiliate TikTok Shop
   ↓ Track via kode unik per KOL di spreadsheet

4. TRACKING (Ongoing)
   ↓ Monitor GMV per KOL setiap Senin
   ↓ Bayar komisi setiap Rabu (H+7 konfirmasi pesanan)
   ↓ Screenshot performa → kirim ke KOL (motivasi)

5. SCALING (Bulan 2-3)
   ↓ KOL dengan views > 10K atau GMV > Rp 500K → boost via TikTok Ads Rp 50-100K
   ↓ KOL dengan 3 bulan konsisten → naikkan komisi 5%
   ↓ Top 10 GMV KOL → feature di IG Story 2bShine
```

### Hook Konten High Converting

1. **"POV: Kulit kusam padahal rajin skincare"**
   → Relatable problem, banyak yang relate, CTR tinggi

2. **"Dermatologist bilang ini bahan aktif paling worth it di harga segini"**
   → Authority + value, edukatif, viral potential tinggi

3. **"Saya challenge pakai 2bShine 30 hari tanpa skincare lain"**
   → Progress video, bikin orang penasaran, completion rate tinggi

4. **"Beli skincare Rp 30rb vs Rp 300rb, bedanya apa sih?"**
   → Perbandingan, engaging, positioning value product

5. **"POV: Temen nanya pakai skincare apa sampe glowing gini"**
   → Aspirational, lifestyle fit, natural endorsement

### Sistem Komisi & Insentif (Revisi — Threshold Realistis)

```
Komisi Base: 10–15% per transaksi via link affiliate

Bonus Milestone (threshold yang achievable):
- Views > 10K dalam 7 hari → Rp 25K cash bonus (motivasi konten)
- GMV Rp 500K–1jt/bulan → +3% bonus komisi
- GMV > Rp 2jt/bulan → Brand Ambassador title + Rp 250K cash
- 3 bulan konsisten posting → Opsi kontrak endorsement (negosiasi)

Non-cash insentif:
- Full PR package gratis setiap bulan (jika GMV > 500K)
- Early access produk baru
- Feature di IG 2bShine Story (top 10 GMV)
```

### Cara Konten Affiliate Bisa Viral + Selling

```
✅ Format: Talking head + close-up skin texture (ASMR skincare)
✅ Sound: Trending audio (update tiap minggu dari brief)
✅ Caption: "Link di bio / TikTok Shop" — jelas
✅ Timing: Upload sore 16.00–20.00 WIB
✅ Engagement bait: "Comment 'GLOW' buat aku kirimin info-nya"
✅ Authenticity: Real skin (jerawat, pori) — jangan over-edit
✅ CTA: Spesifik, clear — "Klik link TikTok Shop sekarang, stok terbatas"

❌ Jangan: Klaim berlebihan, before-after manipulatif, terlalu promotional
```

## 9. RISK & MITIGASI

| Risiko | Kemungkinan | Mitigasi |
|--------|-------------|----------|
| KOL tidak posting dalam 7 hari setelah terima produk | Medium | Send reminder script hari ke-5, follow up DM di hari ke-8 |
| Meta ROAS < 2x di minggu ke-3 | Medium | Ganti creative angle, BUKAN naikkan budget |
| Shopee kehabisan stok saat ads sedang jalan | Low-Medium | Jaga stok minimum 200 unit sebelum launch ads |
| TikTok konten KOL ada klaim berlebihan | Low | Approval flow untuk klaim ingredient/hasil medis |
| Komisi tidak dibayar tepat waktu → KOL tidak aktif | Low | Bayar setiap Rabu, jangan tunggu kumpul banyak |
| Budget Meta habis sebelum akhir bulan | Medium | Set daily cap, monitor setiap hari |
